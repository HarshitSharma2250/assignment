# Areness Backend Assignment

This project is a basic backend API for user registration and login using Node.js, Express, MongoDB, and JWT.

## Live Links
- Backend: https://areness-assignment-wjeu.onrender.com/home
- Frontend: https://arenessharshitmern.netlify.app/

## Features
- User Registration
- User Login with JWT
- Password hashing using bcrypt
- MongoDB integration
- CORS enabled

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
